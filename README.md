# Save Buttons
Save your data without closing the game.

## Features
- A save popup in the main menu with buttons to save your game data, local levels, or mod data without closing the game
- A scrollable list of mods to save data for
- A customizable keybind to open the save popup (Ctrl+Shift+S/Cmd+Shift+S by default) (Requires Custom Keybinds by Geode Team)
